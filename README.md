# Från klassrummen till rymden, en röd tråd från nyfikenhet till framtidens ingenjörer!
BOREALIS är ett projekt för högstadieelever som syftar till att introducera unga till teknik och rymdforskning genom praktiska, rymdnära experiment anpassade till deras kunskapsnivå. Genom att designa och bygga små experiment som skickas upp med stratosfäriska ballonger får eleverna uppleva ett verkligt rymdprojekt, vilket ger dem en första inblick i rymdteknik och ingenjörskonst. Projektet är utformat för att passa den grundläggande förståelse för naturvetenskap som elever på högstadiet har, och fungerar som en inspirerande introduktion till STEM-ämnen och framtida rymdrelaterade projekt.
